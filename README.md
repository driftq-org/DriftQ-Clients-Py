# DriftQ-Clients-Py

Python SDK for **DriftQ** — produce and consume messages, and call admin APIs.

- Minimal, typed Python 3.10+
- Future gRPC integration; current APIs are no-op stubs
- No codegen required for end‑users

---

## Requirements
- Python 3.10+

---

## Install (when published)
```bash
pip install driftq-clients-py
```

For local development:
```bash
pip install -e .
```

---

## Usage (minimal)

> In this initial release, methods are **no‑ops** to keep the SDK importable until gRPC stubs land. See `/examples` for runnable scaffolds.
